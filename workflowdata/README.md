# Workflow Data

This directory contains artifacts generated by GitHub Actions workflows.

## Contents

- **news_output_*.json**: Daily news data fetched from Finnhub API
  - Generated by: `.github/workflows/daily_news.yml`
  - Schedule: Daily at 6 PM New York Time
  - Format: JSON files with date-stamped filenames

## Notes

Files in this directory are automatically generated and committed by GitHub Actions workflows.
